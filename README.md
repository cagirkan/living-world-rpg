# Living World RPG

## Açıklama
Açık dünya, fantastik bir RPG projesi. Oyuncu farklı karakter sınıflarından birini seçerek keşif, savaş ve diplomasi mekaniklerini deneyimler. Envanterdeki eşyalar kullanım sırasında seviye atlar ve sürpriz yetenek/bonuslar açılır.

## Başlarken

### Gereksinimler
- Unity Hub + Unity 2024.x LTS (3D URP)
- Visual Studio Community veya JetBrains Rider
- Git

### Kurulum
1. Depoyu klonlayın:
   ```bash
   git clone https://github.com/<kullanici>/living-world-rpg.git
   ```
2. Unity Hub’dan proje klasörünü seçerek açın.
3. Unity Package Manager’dan gerekli paketleri yükleyin (`Cinemachine`, `Input System`, vb.).
